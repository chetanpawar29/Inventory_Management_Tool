# 📦 Inventory Management Tool
The Inventory Management Tool is a robust & efficient web application designed to simplify inventory operations.It empowers inventory managers to seamlessly track, manage, & issue raw and processed materials, ensuring smooth business operations. 🚀

# ✨ Key Features
## 🔑 Authentication & Security
- Secure login functionality for inventory managers.
- Password change option to ensure account safety.
## 📋 Raw Material Management
➕ Add new raw materials with detailed information.

🔍 View and update raw material quantities.

❌ Delete raw materials when no longer needed.

## 🏭 Processed Material Management
➕ Add processed items effortlessly.

🔍 View and update processed item details.

❌ Delete processed items with ease.
## 🛒 Material Issuance
✅ Issue raw materials and processed items for use.

📜 View detailed logs for issued raw and processed items.

## 📊 Availability Reports
📌 Identify unavailable raw materials at a glance.

📌 Check for processed items that are out of stock.

## 🛡️ Enhanced Usability
- User-friendly design for quick navigation.
- Real-time updates on inventory changes.
  
📌 Note: The system restricts item issuance if the quantity of raw or processed materials is zero, ensuring accurate inventory tracking.

## 💻 Tech Stack
🌐 Frontend: HTML, CSS, JavaScript

⚙️ Backend: Spring Framework, Hibernate, JSP

🛢️ Database: MySQL (XAMPP)

📚 Database Schema

📂 Database Name: inventoryDB
## 🚀 Getting Started
- Clone the repository to your local machine.
- Set up the database using the provided schema.
- Configure the backend with your database credentials.
- Run the application and access it via your preferred browser.
